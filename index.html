Generate a CO₂ Emissions Shield for Your Github Projects.

The CO₂ Shield Generator lets you easily display the CO₂ emissions of a target URL in your GitHub project with a markdown badge, powered by shields.io. Using the co2.js library from The Green Web Foundation, it calculates the emissions for the given URL. Just enter the URL, click “Generate Shield,” and copy the markdown and paste into your project documentation. Learn more about the project on the CO₂ Shield Github page.
